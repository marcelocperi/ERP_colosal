Guía: Cómo conectar el Sistema a AFIP

Ejecute este comando en la terminal del ERP:

python multiMCP/util_generar_afip.py

Esto creará dos archivos en la carpeta del proyecto:

  • privada.key (NO COMPARTIR - Llave secreta)
  • pedido.csr (Súbalo a AFIP)

  1. Ingrese a afip.gob.ar -> Administración de Certificados Digitales.
  2. Cree un nuevo Alias y suba el archivo pedido.csr.
  3. Descargue el certificado .crt.

  1. Vaya a Administrador de Relaciones -> Nueva Relación -> AFIP -> Webservices.
  2. Busque: Consulta de Padrón Alcance 13 y/o Facturación Electrónica.

Pegue el contenido de privada.key y el .crt en la pantalla de Configuración Fiscal de este sistema.
Estos padrones se procesan localmente subiendo los archivos de cada ente recaudador.
Jurisdicción Método / Formato Datos Específicos
CABA (AGIP) TXT (Posiciones fijas) Alícuotas de Percepción y Retención personalizadas.
Bs. As. (ARBA) TXT en ZIP (Masivo) Cálculo de alícuota sobre CUITs inscriptos en la provincia.
Convenio Multi. AFIP API (RUT) Detección automática de jurisdicciones donde tributa el CUIT.
Santa Fe (API) TXT / CSV (Delimitado) Padrón de Sujetos Exentos y Alícuotas de Retención/Percepción.
Córdoba (DGR) CSV (Punto y Coma) Base de datos de contribuyentes locales y alícuotas según riesgo.
Misiones (ATM) Múltiples TXT (Regímenes) Incluye exclusiones temporales y alícuotas incrementales.
Otras (Misc) Formato Genérico Soporta mapeo de columnas para Tucumán, Salta y Mendoza.
Instructivo de Procesamiento:

1. Descargue el padrón desde la web de rentas correspondiente.
2. Use el botón "Importar Padrón" en esta página.
3. El ERP lee el archivo línea por línea y actualiza masivamente las tablas impositivas.
