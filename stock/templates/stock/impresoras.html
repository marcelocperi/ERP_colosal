Impresoras de Etiquetas

Configuración de perfiles térmicos — Soporta Zebra, TSC, Brother (USB, Red TCP/IP o QZ Tray)

Modo A — Diálogo Navegador

El navegador abre la ventana nativa de impresión. Compatible con USB, Red y Bluetooth. El usuario elige la impresora la primera vez y puede fijarla como default en el sistema.

Modo B — IP de Red (ZPL directo)

El servidor envía ZPL crudo por socket TCP al puerto 9100 de la impresora. Requiere Ethernet / WiFi en la impresora. Sin popups, impresión automática desde el servidor.

Modo C — QZ Tray (USB / Red)

Agente local instalado en la PC de depósito. Habla con impresoras USB o de red sin diálogos. Ideal para operativas de alto volumen. Requiere instalar QZ Tray en cada terminal.
