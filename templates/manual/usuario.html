Manual de Usuario

1. Acceso y Seguridad

Para ingresar al sistema, utilice sus credenciales asignadas. El sistema cuenta con roles diferenciados (Administrador, Bibliotecario, Usuario).

Importante: Si olvida su contraseña, utilice la opción "Recuperar Contraseña". Recibirá un código temporal en su correo electrónico.

2. Catálogo de Libros

La pantalla Libros permite administrar el acervo bibliográfico.

  • Búsqueda: Utilice la lupa en la barra superior o el botón "Opciones de Búsqueda" para filtrar por Título, Autor, ISBN o Género.
  • Nuevo Libro: Haga clic en "Agregar Libro" para registrar un nuevo título. Se validará automáticamente que el ISBN no esté duplicado.
  • Edición: Seleccione un libro de la grilla para modificar sus datos o ver su historial.

3. Gestión de Préstamos

El núcleo operativo de la biblioteca.

Realizar un Préstamo

  1. Navegue a la sección Préstamos.
  2. Haga clic en "Nuevo Préstamo".
  3. Seleccione el Usuario y el Libro. El sistema validará automáticamente:
    • Que el usuario no tenga deuda pendiente.
    • Que exista stock disponible.
    • Que el libro no tenga reservas activas de otros usuarios.
  4. Confirme la operación. El usuario recibirá un email de notificación.

Devolución

Desde la lista de préstamos activos, localice el préstamo y haga clic en Devolver. Si el libro tiene reservas pendientes, el sistema le alertará y notificará automáticamente al siguiente usuario en la cola.

4. Gestión de Reservas

Cuando no hay ejemplares disponibles, se puede registrar una reserva.

  • Registro: Si intenta prestar un libro sin stock, el sistema le ofrecerá la opción de Reservar.
  • Notificación: Al registrarse, el usuario recibe un correo confirmando su posición.
  • Disponibilidad: Cuando se devuelva un ejemplar, el sistema detectará la reserva y enviará un correo de "Libro Disponible" con validez de 48 horas.

5. Inventario y Stock

Control detallado de ejemplares físicos.

La pantalla de Inventario muestra una vista consolidada de:

  • Stock Total (Ejemplares físicos registrados).
  • Stock Disponible (En estantería).
  • Prestados (Fuera de la biblioteca).
  • Reservados (Comprometidos).
