# Lógica de Selección de Comprobante (BillingService)

1. Verificar condicion_iva de Empresa.
2. Cruzar con tipo_responsable del Cliente.
3. Si Emisor='RI' y Receptor='RI' -> FACTURA A (001).
4. Si Receptor='CF' o 'MT' -> FACTURA B (006).
5. Recalcular Percepciones IIBB por jurisdicción.
