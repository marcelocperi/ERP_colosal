# Algoritmo de Explosión Recursiva de Costos

1. Buscar receta activa en cmp_recetas_bom.
2. Si componente es Producido -> Llamar get_industrial_cost().
3. Si componente es Insumo -> Buscar Sourcing HABITUAL.
4. Sumar Gastos en cmp_articulos_costos_indirectos.
5. Aplicar Margen Proyectado -> precio_sugerido.
